Baca terlebih dahulu langkah-langkah di bawah ini:

1. Buatlah folder css, folder img, dan folder class
    di dalam folder kelas yang sudah dibuat sebelumnya
    dalam /var/www/kelasxxxxx

2. Kemudian ketik masing-masing code menggunakan editor
    dan simpan file dalam masing-masing folder

3. Untuk file 'customer_gui.php' disimpan dalam root folder
    untuk file class 'customer.php' disimpan dalam folder 'class'
    untuk file gambar 'customer.png' disimpan dalam folder 'img'
    untuk file css 'customer.css' disimpan dalam folder 'css'
